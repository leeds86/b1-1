# 모델 성능 비교
📋 회의록 요약 LLM 선정을 위한 4대 평가 축
1. 내용의 정확성 및 수치 정밀도 (Accuracy & Precision)
정의: 회의 중 언급된 핵심 결정 사항과 수치(날짜, 금액, 인원 등)를 왜곡 없이 반영하는가?
체크 포인트:
회의 중 변경된 최종 의사결정(예: 15일 → 22일 변경)을 정확히 인지했는가?
금액(5,000만 원)이나 비율(15% 하락) 등 숫자를 틀리지 않았는가?
발언자와 발언 내용을 매칭하는 과정에서 오류(환각 현상)가 없는가?
2. 실행 과제(To-Do) 추출의 구체성 (Action Item Extraction)
정의: 회의 후 후속 조치를 위해 '누가, 무엇을, 언제까지' 해야 하는지 명확히 뽑아내는가?
체크 포인트:
담당자(누가), 업무 내용(무엇을), 마감 기한(언제까지)이 모두 포함되었는가?
단순 언급된 내용과 실제 '할 일'을 구분해낼 수 있는가?
누락된 할 일은 없는가?
3. 논리적 구조화 및 가독성 (Structuring & Readability)
정의: 복잡한 대화 내용을 비즈니스 문서 형식에 맞춰 보기 좋게 정리하는가?
체크 포인트:
개요-결정사항-할 일 등 논리적인 구조를 갖추었는가?
불렛포인트, 표, 굵은 글씨 등을 적절히 사용하여 한눈에 들어오는가?
비즈니스 환경에 적합한 격식 있는 말투(한국어 자연스러움)를 사용하는가?
4. 문맥 유지 및 관계 파악 (Context & Relationship)
정의: 대화의 흐름을 놓치지 않고, 인물 간의 관계나 논쟁의 인과관계를 이해하는가?
체크 포인트:
A의 제안에 대한 B의 반대 의견과 그에 따른 합의 과정을 잘 요약했는가?
긴 대화(Long Context) 속에서도 앞부분의 맥락을 끝까지 유지하는가?
단순 요약을 넘어 회의의 '목적'에 부합하는 결론을 도출했는가?
